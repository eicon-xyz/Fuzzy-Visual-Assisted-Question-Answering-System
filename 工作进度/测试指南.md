## 七、常见问题

| 问题 | 解决 |
|------|------|
| `No module named 'fastapi'` | `pip install fastapi uvicorn` |
| `No module named 'cv2'` | `pip install opencv-python-headless` |
| A 端启动后立刻退出 | 检查 `HAJIMI_UI/server/.env` 是否存在 |
| ASR 录音失败 | `pip install pyaudio` |
| TTS 声音走错设备 | 右键任务栏喇叭图标 → 选择目标输出设备 |
| Web 面板图表不显示 | 检查 `web-admin/node_modules` 是否存在，执行 `npm install` |
| 端口 8010 被占用 | 见下方"端口冲突处理" |

### 端口冲突处理

启动 A 端时报错 `[WinError 10048] 通常每个套接字地址只允许使用一次`，说明上次启动的进程没关。

```bash
# 1. 找到占用的 PID
cmd //c "netstat -ano | findstr :8010 | findstr LISTENING"
# 输出示例：TCP  127.0.0.1:8010  ...  LISTENING  18956
#                                                      ^^^^^ 这个就是 PID

# 2. 杀掉它
cmd //c "taskkill /PID 18956 /F"

# 3. 重新启动
cd HAJIMI_UI
python -m uvicorn server.main:app --host 127.0.0.1 --port 8010
```
