### 2. 审计代理 HTTP 上报联调准备 ✅

**文件**：`client/audit/audit_agent.py`

- 批量 POST `/api/audit/report` 请求体对齐 A-C 接口契约 §3.1
- 请求格式：`{ client_id, batch: [...] }`
- 响应处理：`received` 确认后删除本地记录；`server_queue_depth > 100` 时自动减缓上报频率
- 指数退避重试链（1min/5min/15min/1h），超 3 次写 `fallback.log`

### 3. 配置轮询器联调准备 ✅

**文件**：`client/config/config_poller.py`

- 定时轮询 `GET /api/config/pull`，携带 `X-Client-Version` + `If-None-Match`
- 304 响应 → 本地缓存继续工作
- 配置变更 → 版本号比对 → `on_config_changed` 回调通知
- 轮询间隔支持服务端动态下发调整（5~1440min）

### 4. ASR 引擎多方案测试 ✅

- Vosk 离线小模型：准确度偏低，适合无网络隐私场景
- **Google Web Speech（当前默认）**：免费在线引擎，中文识别准确度明显优于 Vosk 小模型
- 百度语音 API 预留：中文识别最优，免费 5 万次/天，备选
